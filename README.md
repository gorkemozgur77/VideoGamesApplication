# Video Games Application  
  
MVVM tasarım kalıbını takip ederek sunucudan veri almak ve yerel veritabanı işlemleri ile ekranda listelemek için bir Kotlin uygulaması.  
  
### Kullanılan kütüphaneler  
  
* MVVM tasarım kalıbı.  
* Retrofit  
* Room  
* Firebase Analytics   
* Coroutines
